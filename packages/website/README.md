# RNRepo Website

This is the website for RNRepo, built with [Astro](https://astro.build).

## Development Setup

### Install Dependencies

```bash
bun install
```

### Run Development Server

```bash
bun run dev
```

The site will be available at `http://localhost:4321`.

### Build for Production

```bash
bun run build
```

### Preview Production Build

```bash
bun run preview
```
